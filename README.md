# screenrec
This is a pyhton screen recorder
### Prerequisite
* Python3

### Instalation
1. Clone the repo
   ```sh
   git clone https://github.com/screenrec/pa.git
   ```
2. Run install.py (only once)
  ```sh
  pyhton3 install.py
  ```

   
### Run the code for second time
   ```sh
   python3 main.py
   ```
